#  Product Service — GraphQL API Specification

##  Описание
Спецификация GraphQL API для сервиса управления товарами. Контракт разработан на основе бизнес-требований к Q2 и описывает CRUD-операции, структуру данных, форматы ответов и правила интеграции.

##  Структура

## Реализованные требования
- [x] Полный CRUD для товаров (`Query` + `Mutation`)
- [x] Обязательные и опциональные поля (описание товара заполняется не всегда)
- [x] Даты в формате ISO 8601 (`String!` в схеме, валидация на бэкенде)
- [x] Дедупликация при создании через обязательный параметр `idempotencyKey`
- [x] Кастомный тип ответа на удаление `DeleteProductPayload` (`success: Boolean!`, `errorMessage: String`)
- [x] Единая точка входа `/graphql` (стандарт компании)
