# 📈 Market Data & Charts — GraphQL API Specification

##  Описание
Спецификация GraphQL API для сервиса получения рыночных данных: котировок по тикерам, исторических данных для графиков (линейных и свечных) и подписок на обновления в реальном времени. Контракт разработан для замены вендорской системы и обеспечивает гибкий доступ к финансовым инструментам.

##  Реализованные требования
- [x] Получение цены по тикеру (опциональная дата → последняя цена)
- [x] Получение данных для графика (тикер, таймфрейм, опциональный период → дефолт 1 месяц)
- [x] Подписка на обновления графика в реальном времени (`Subscription`)
- [x] Поддержка линейных и свечных графиков (OHLC + lastPrice)
- [x] Опциональные поля: `close` (свеча может не закрыться), `indicators`
- [x] Индикаторы: поддержка массива значений `[Float!]!` (унификация для 1..N значений)
- [x] Даты в формате ISO 8601 (`scalar ISO8601DateTime`)
- [x] Единая точка входа `/graphql`, HTTP всегда `200 OK`
