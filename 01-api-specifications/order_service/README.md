# Order Service API

gRPC/Protobuf контракт для управления заказами с поддержкой gRPC Gateway (HTTP/REST).

## 📋 Требования

### Бизнес-требования
- ✅ Просмотр существующего заказа по ID
- ✅ Просмотр всех заказов с опциональной пагинацией
- ✅ Создание нового заказа
- ✅ Удаление заказа

### Атрибуты заказа
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `int64` | Уникальный идентификатор заказа |
| `user_id` | `int64` | ID связанного пользователя |
| `legacy_user_id` | `int64` | ⚠️ Устаревший интеграционный ID (`deprecated`) |
| `products` | `repeated Product` | Список товаров в заказе |
| `total_price` | `double` | Итоговая стоимость заказа |
| `delivery_type` | `Delivery` | Тип доставки (enum) |

### Атрибуты товара
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `int64` | Уникальный идентификатор товара |
| `name` | `string` | Название товара |
| `description` | `string` | Описание товара |
| `price` | `double` | Цена товара |

### Типы доставки (enum `Delivery`)
```protobuf
enum Delivery {
  UNKNOWN = 0;           // Неизвестный тип
  FAST_DELIVERY = 1;     // Быстрая доставка
  REGULAR_DELIVERY = 2;  // Обычная доставка
  PICKUP = 3;            // Самовывоз
}
